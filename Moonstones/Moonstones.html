RPG Maker Privacy Policy

This game is made with RPG Maker MV and therefore it is javascript and HTML code which is compiled to an APK. Before this, the developer has access to the manifest but at he/she is not allowed to define permissions for the app. This means, the third party software, which is used to compile the Javascript/HTML code adds permission to the application during the compilation. This permissions are:

  • Access to media
  • Access to camera
  • Access to microphone
  • Access to WLAN connection information

This app does nothing with your media or any data from camera, microphone or WLAN. These permissions are added from the tools which where used for compilation.
